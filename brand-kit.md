# Brand Kit — The Whole Again Initiative

## Colors (Desert Rose Theme)

| Name | Hex | Usage |
|------|-----|-------|
| **Dusty Rose** | `#d4a5a5` | Soft primary, accent borders, active nav indicator |
| **Clay** | `#b87d6d` | Earthy accent — labels, links, nav text |
| **Sand** | `#e8d5c4` | Warm neutral backgrounds, text on dark sections |
| **Deep Burgundy** | `#5d2e46` | Primary headings, buttons, dark section backgrounds |
| **Petal** | `#f5ece6` | Light section backgrounds |
| **Warm White** | `#fdf8f5` | Page background |
| **Muted Rose** | `#c49090` | Defined but currently unused |
| **Text Dark** | `#3a1e2e` | Body text |
| **Footer Dark** | `#2a1020` | Footer background |

## Typography

| Role | Font | Weight(s) | Details |
|------|------|-----------|---------|
| **Display / Headings** | Cormorant Garamond | 300, 400, 500, 600 (+ italic 300, 400) | Elegant serif, used at large sizes |
| **Body** | EB Garamond | 400, 500 (+ italic 400) | 18px base, 1.75 line-height |
| **UI / Labels / Nav** | Jost | 300, 400, 500 | Sans-serif, uppercase, letter-spacing 0.14–0.22em |

All served via Google Fonts.

## Key Design Elements

- **Buttons**: `.btn-primary` — filled deep burgundy, inverts to outlined on hover. `.btn-secondary` — outlined dusty rose.
- **Section labels**: Small Jost uppercase eyebrow text in clay
- **Page banners**: Gradient from petal to sand
- **Dark sections**: Deep burgundy background with sand text and dusty rose accents
- **Scroll animations**: `.reveal` class with fade-up via IntersectionObserver
- **Nav**: Fixed, frosted glass effect (blur + semi-transparent warm white)

## Brand Tone

- Warm, empathetic, elegant
- Serif-forward for trustworthiness and sophistication
- Muted earth/rose palette — avoids clinical feel, conveys care and warmth
